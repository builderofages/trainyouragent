## Next Steps for User

1. **Create 5 New Industry Tags in Apollo.io:**
   - Go to Settings → Labels → Create Label
   - Add: Spas, Hotels, Automotive, Real Estate, Solar
   - Set type to "Contact"

2. **Create 5 New Landing Page Tags:**
   - Add: Spas Landing, Hotels Landing, Automotive Landing, Real Estate Landing, Solar Landing

3. **Update `industry` Custom Field Dropdown:**
   - Go to Settings → Custom Fields → Edit "industry" field
   - Add 5 new options: Spas, Hotels, Automotive, Real Estate, Solar

4. **Test New Landing Pages:**
   - Submit test leads from /spas, /hotels, /automotive, /real-estate, /solar
   - Verify correct industry tags applied
   - Verify custom field data populated
